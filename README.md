# 🚀 Bootcamp de Programación

Bienvenido a este repositorio. Aquí encontrarás todo el código, ejemplos y ejercicios que se desarrollan durante el bootcamp de programación.

## 📚 Descripción

Este repositorio contiene el material práctico utilizado en las clases:

- Ejemplos explicados paso a paso
- Ejercicios para reforzar conceptos
- Retos propuestos durante el bootcamp
- Soluciones y buenas prácticas

El objetivo es que los alumnos puedan revisar, practicar y profundizar en los temas vistos en cada sesión.

---

## 🧠 Metodología

El bootcamp está enfocado en el aprendizaje práctico:

- 💻 Código en vivo durante las clases
- 🧩 Ejercicios progresivos
- 🔁 Repetición y refuerzo de conceptos clave
- 🚀 Desarrollo de proyectos reales
