LISTAS
Existen 3 tipos de lista:
- UL => Unordered list => Lista desordenada
- OL => Order List => Lista ordenada
- DL => Description List => Lista de definición (FAQs, metadatos, glosarios)

TABLAS
- <table> => Contenedor de toda la tabla
- <tr> => table row => fila de la tabla
- <th> => table header => celda de encabezado (negrita y centrado por defecto)
- <td> => table data => celda de datos
