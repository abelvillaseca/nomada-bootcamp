IMÁGENES
- Traen un archivo externo hacia tu página
- src = source => Ruta o fuente de tu archivo de imagen.
- alt = alternativo => Texto alternativo que describe la imagen
- width = Ancho en píxeles (opcional)

FORMATOS DE IMÁGENES
- JPG => Fotos, sin transparencia y buen peso
- PNG => Logos, gráficos con transparencia
- WebP => Moderno, liviano. Es el más recomendado
- SVG => Vectorial: íconos y logos que escalan sin pixelarse
