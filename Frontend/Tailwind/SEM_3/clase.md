EL MODELO DE CAJA (BOX MODEL) CON UTILIDADES

- Tamaños (dimensiones)

 w-64 => ancho fijo (escala de Tailwind)
 h-32 => alto fijo
 w-full => 100% contenedor padre
 w-1/2 => la mita del contenedor padre
 max-w-sm / md / lg / xl 
 min-h-screen => alto mínimo igual a la altura visible de la ventana

- bordes

border => 1px
border-2 => más grueso
border-slate-200 => color
border-t => solo arriba

- radios
rounded y sus variantes

- sombras
shadow-sm / shadow / shadow-md / shadow-lg

- fondos
