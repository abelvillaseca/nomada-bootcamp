En un proyecto nuevo, crea tres scripts: saludar que imprima tu nombre, fecha que muestre la fecha del sistema, y todo que ejecute ambos en orden. Ejecuta npm run todo.

==============================================================

Crea un script deploy y, usando hooks, haz que automáticamente antes imprima "Verificando..." (predeploy) y después imprima "Desplegado con éxito" (postdeploy). Al ejecutar npm run deploy deben verse las tres líneas en orden.

Pista: recuerda los prefijos pre y post.
